Miwok App
===================================

This app displays lists of vocabulary words for the user to learn the Miwok language. This app is built using Java in Android Studio.

## **Topics Covered**

Implemented
- MediaPlayer
- Adapter
- ListView
- View recycling
- Audio Focus
- onClickListener
- OnAudioFocusChangeListener
